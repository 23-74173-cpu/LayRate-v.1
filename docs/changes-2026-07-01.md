## Cage Overview (Dashboard)

### Gap fix — unplaced pills
- **`resources/views/dashboard.blade.php`**: Replaced `gap-2` workaround with intentional corner-overspill badge. Badge now sits `absolute -top-2 -right-2 z-10` at **all** breakpoints (desktop + mobile), overlapping the pill's top-right corner by 8px.

### Mobile grid reduction — placed tiles
- Added `<style>` block forcing `.cage-grid` to `repeat(2, minmax(0, 1fr))` below 640px (`max-width: 639px` via `!important` override). At 375–390px viewports each card gets ~160px width, enough for code + subtitle + overspilling badge to render legibly.

### Badge position standardization
- Both placed grid tiles and unplaced pills now use the identical overspill pattern: `absolute -top-2 -right-2 z-10` (`sm:static` removed entirely, no breakpoint-specific positioning).
